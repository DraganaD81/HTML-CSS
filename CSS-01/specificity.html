Specificity

Specificity is weight that is applied to a given CSS declaration, determined by number of each selector type in the maching selector. When multiple declarations have equal specificity, the last declaration found in the CSS is applied to the element. Specificity only applies when the same element is targeted by multiple decarations. As per CSS rules, directly targeted elements will alwas take precedence over rules witch an element inherits from its ancestor.
